# Installation

```
$ npm i db-migrate -g
$ git clone https://github.com/siteslave/test-db-migrate.git
$ cd test-db-migrate
$ npm i
```

# Configure
Edit file `database.json`

# Running

```
$ db-migrate up:v1
$ db-migrate reset:v1
```
